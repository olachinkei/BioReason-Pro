# BioIT World Talk Deck

Slidev deck for the BioIT World presentation:

**LLMOps for Drug Discovery Systems: Governing Models, Agents, and Evaluation**

## Setup

```bash
cd presentations/bioitworld
npm install
```

## Development

```bash
npm run dev
```

Default local URL: `http://localhost:3030`

## Build

```bash
npm run build
```

Static output is generated in `dist/`.

## Export PDF

```bash
npm run export
```

If needed:

```bash
npx playwright install chromium
```
